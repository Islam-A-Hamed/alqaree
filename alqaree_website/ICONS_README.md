# أيقونات الموقع - Icons Guide

## الأيقونات الحالية

تم إنشاء أيقونات SVG بسيطة للموقع تحتوي على:
- شعار "القارئ" باللغة العربية
- خلفية خضراء متدرجة
- تصميم دائري أنيق

## الملفات الموجودة

- `icon-192x192.svg` - أيقونة بحجم 192x192 بكسل
- `icon-512x512.svg` - أيقونة بحجم 512x512 بكسل

## لإنشاء أيقونات أفضل

### باستخدام أدوات عبر الإنترنت:
1. **Canva**: https://www.canva.com/
2. **Favicon.io**: https://favicon.io/
3. **RealFaviconGenerator**: https://realfavicongenerator.net/

### التصميم المقترح:
- استخدم اللون الأخضر (#28a745) كلون رئيسي
- أضف رمز القرآن أو الهلال كعنصر تصميمي
- استخدم خط عربي جميل لكتابة "القارئ"
- احرص على أن يكون التصميم واضحاً على جميع الأحجام

### الأحجام المطلوبة:
- 192x192 بكسل (للـ manifest)
- 512x512 بكسل (للـ manifest)
- 32x32 بكسل (favicon)
- 180x180 بكسل (Apple touch icon)

### كيفية الاستبدال:
1. أنشئ الأيقونات باستخدام الأدوات أعلاه
2. احفظها بصيغة PNG مع خلفية شفافة
3. استبدل ملفات SVG الحالية بالملفات الجديدة
4. حدث `manifest.json` لتشير إلى الملفات الجديدة

## ملاحظات مهمة:
- الأيقونات الحالية هي مؤقتة وستعمل بشكل جيد
- يُفضل استخدام SVG لأنه يدعم التكبير بدون فقدان الجودة
- تأكد من أن الأيقونات تظهر بوضوح على جميع الخلفيات
