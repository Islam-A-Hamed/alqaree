# موقع القارئ - Alqaree Website

## نظرة عامة على المشروع

موقع القارئ هو موقع ويب باللغة العربية يهدف إلى تقديم المحتوى الإسلامي بشكل منظم وسهل الوصول. يحتوي الموقع على تلاوات قرآنية، حكم ومواعظ، والقرآن الكريم.

## التقنيات المستخدمة

- **اللغة البرمجية**: PHP
- **قاعدة البيانات**: MySQL
- **التصميم**: CSS3 مع Google Fonts (Noto Naskh Arabic)
- **التفاعل**: JavaScript (ES6+)
- **الخادم**: Apache (XAMPP)

## هيكل المشروع

```
alqaree_website/
├── index.php                 # الصفحة الرئيسية
├── tilawat.php              # صفحة التلاوات
├── hekum.php                # صفحة الحكم والمواعظ
├── quran.php                # صفحة القرآن الكريم
├── css/
│   └── style.css            # ملف التصميم الرئيسي
├── js/
│   └── script.js            # ملف JavaScript للتفاعل
├── includes/
│   ├── header.php           # رأس الصفحة
│   ├── footer.php           # تذييل الصفحة
│   ├── config.php           # إعدادات PHP
│   └── db_connect.php       # اتصال قاعدة البيانات
├── admin/
│   ├── manage-tilawat.php  # إدارة التلاوات (المتقدمة)
│   └── manage-hekum.php  # إدارة المواعظ (المتقدمة)
├── sql/
│   └── database.sql         # ملف قاعدة البيانات
└── images/                  # مجلد الصور (فارغ حالياً)
```

## قاعدة البيانات

### الجداول الرئيسية

1. **recitations** - التلاوات القرآنية
   - id (Primary Key)
   - title (عنوان التلاوة)
   - surah_name (اسم السورة)
   - reciter_name (اسم القارئ)
   - video_duration (مدة الفيديو)
   - publish_date (تاريخ النشر)
   - description (الوصف)
   - youtube_embed_code (كود تضمين يوتيوب)
   - category_id (Foreign Key)

3. **sermons** - الحكم والمواعظ
   - id (Primary Key)
   - title (عنوان الموعظة)
   - speaker_name (اسم المتحدث)
   - video_duration (مدة الفيديو)
   - publish_date (تاريخ النشر)
   - description (الوصف)
   - youtube_embed_code (كود تضمين يوتيوب)
   - category_id (Foreign Key)

4. **quran_verses** - آيات القرآن الكريم
   - id (Primary Key)
   - surah_number (رقم السورة)
   - ayah_number (رقم الآية)
   - ayah_text (نص الآية)
   - juz_number (رقم الجزء)
   - page_number (رقم الصفحة)

## الميزات الرئيسية

### 1. الصفحة الرئيسية
- تصميم جذاب مع قسم البطل (Hero Section)
- روابط سريعة للتلاوات والقرآن والمواعظ
- تصميم متجاوب للهواتف والأجهزة اللوحية

### 2. صفحة التلاوات
- عرض التلاوات في شبكة منظمة
- كل تلاوة تحتوي على:
  - فيديو من يوتيوب
  - معلومات القارئ والسورة
  - المدة وتاريخ النشر
- نافذة منبثقة لعرض التفاصيل الكاملة
- ترتيب حسب تاريخ النشر (الأحدث أولاً)

### 3. صفحة الحكم والمواعظ
- عرض المواعظ في شبكة منظمة
- كل موعظة تحتوي على:
  - فيديو من يوتيوب
  - معلومات المتحدث
  - المدة وتاريخ النشر
- نافذة منبثقة لعرض التفاصيل الكاملة

### 4. صفحة القرآن الكريم
- مُعدة لعرض المصحف الشريف
- (تحتاج لتطوير إضافي)

### 5. لوحة الإدارة
- **إدارة التلاوات**: إضافة، تعديل، حذف التلاوات
- **إدارة المواعظ**: إضافة، تعديل، حذف المواعظ
- واجهة سهلة الاستخدام مع نماذج وجداول منظمة
- رسائل تأكيد للعمليات الناجحة والأخطاء

## التثبيت والتشغيل

### المتطلبات
- XAMPP أو WAMP Server
- متصفح ويب حديث
- اتصال بالإنترنت (لتحميل خطوط Google Fonts)

### خطوات التثبيت

1. **تحميل المشروع**
   ```
   قم بنسخ مجلد المشروع إلى مجلد htdocs في XAMPP
   ```

2. **إنشاء قاعدة البيانات**
   ```sql
   -- في phpMyAdmin أو أداة إدارة MySQL:
   -- قم بتنفيذ محتوى ملف sql/database.sql
   ```

3. **تكوين قاعدة البيانات**
   - افتح `includes/db_connect.php`
   - تأكد من صحة البيانات:
     ```php
     $servername = "localhost";
     $username = "root";     // اسم المستخدم
     $password = "";         // كلمة المرور (عادة فارغة في XAMPP)
     $dbname = "alqaree_website";
     ```

4. **تشغيل الخادم**
   - شغل Apache و MySQL في لوحة تحكم XAMPP
   - افتح المتصفح واذهب إلى: `http://localhost/alqaree_website`
